# Cloud Bridge v0.1.0

Cloud Bridge v0.1.0 is a deterministic, federated coordination substrate.

## Included
- Immutable envelopes with schema negotiation
- Deterministic routing with bounded propagation
- Loop prevention and throttling
- Read-only federation substrate
- Read-only connector interfaces
- Non-invasive observability and audit logging
- Substrate-only load simulation

## Explicitly Excluded
- Autonomous execution
- Write-to-world operations
- Background schedulers
- Operator layer behavior

This release is safe by construction and fail-closed by default.
